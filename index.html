Dungeon Divers is a replica of the classic Venture game by Caleco, featuring intricate dungeon layouts and challenging gameplay.

Playtime: Approximately 5-10 minutes

Group Size: 4 members

Technologies Used: Blender, FLECS (Entity Component System), OpenGL, GitHub

Role: Enemy AI/Gameplay Programmer

Development Time: Approximately 3 months

Responsibilities: Developed enemy AI and managed interactions using FLECS ECS. Worked with OBBs in Blender and C++ to ensure accurate collisions for enemies, the player, and environmental objects.

Playtime: Approximately 10-15 minutes

Group Size: 5 members

Technologies Used: Unreal Engine 5.2, C++, Perforce

Group Size: 4 members

Technologies Used: Unity, C#, GitHub
